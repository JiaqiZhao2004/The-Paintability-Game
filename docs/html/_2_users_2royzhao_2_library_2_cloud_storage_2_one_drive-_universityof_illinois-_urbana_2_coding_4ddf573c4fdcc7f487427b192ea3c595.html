Calculates the position of a node in a circular layout.

Calculates the position of a node in a circular layout. @function calculateNodePositions

Parameters
{number}index - The index of the node in the list of nodes.
{number}num_nodes - The total number of nodes in the graph.
Returns
{Object} An object containing the x and y coordinates of the node.
  • Positions nodes evenly spaced around a circle.
  • The circle is centered at (radius, radius), and the radius can be customized.
  • The angle for each node is calculated as (2 * Math.PI * index) / num_nodes, ensuring equal spacing.
const position = calculateNodePositions(0, 5);
console.log(position); // { x: 300, y: 150 }
